# Ghosty Words 

**Ghosty Words** este un joc educațional interactiv conceput pentru a ajuta utilizatorii să stăpânească verbele și timpurile limbii engleze(Past Simple, Present Continuous) într-un mod distractiv și alert. Jucătorii intră în rolul unei vrăjitoare puternice care trebuie să se apere de fantomele care se îndreaptâ spre ei cu verbele in limba română.

Jocul îți testează cunoștințele de gramatică și viteza de reacție prin două moduri principale:

* Fantomele care poartă verbe în engleză se apropie de vrăjitoare. Pentru a le învinge, trebuie să **tastezi** sau să **pronunți** forma corectă a verbului. Dacă o fantomă ajunge la vrăjitoare, pierzi una dintre cele **3 vieți**.
*  Un nivel în care trebuie să completezi propoziții folosind forma corectă a verbului în contextul potrivit.

## Tehnologii Folosite

* **Backend** Java
